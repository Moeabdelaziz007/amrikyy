# Quantum AI Nexus

A cutting-edge quantum AI portfolio application built with React and featuring a futuristic "GenZ Cyber Neon" aesthetic.

## 🚀 Features

- **Quantum Dashboard**: Real-time quantum metrics and AI insights
- **Social Media Analytics**: AI-powered social analytics with quantum visualization
- **Quantum Features**: Interactive quantum mechanics demonstrations
- **AI Chatbot**: Quantum-themed intelligent assistant
- **Space Crypto Background**: Dynamic particle system with quantum entanglement effects
- **Responsive Design**: Mobile-first responsive layout
- **Glassmorphism UI**: Modern glass effect design elements
- **Neon Animations**: Dynamic quantum-themed animations and effects

## 🎨 Design Philosophy

The application follows a **"GenZ Cyber Neon"** aesthetic with:

- **Color Palette**: Neon Green (#39FF14), Neon Blue (#00d4ff), Neon Purple (#b600ff)
- **Typography**: Modern sans-serif fonts with futuristic styling
- **Animations**: Quantum-inspired particle effects and smooth transitions
- **Glassmorphism**: Frosted glass effect with backdrop blur
- **Quantum Elements**: Scientific accuracy meets visual appeal

## 🛠️ Tech Stack

- **React 18**: Modern React with hooks and functional components
- **Vite**: Fast build tool and development server
- **CSS3**: Advanced animations, glassmorphism, and responsive design
- **Canvas API**: Interactive particle background system
- **Modern JavaScript**: ES6+ features and async/await

## 📦 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Moeabdelaziz007/amrikyy-ai.git
   cd amrikyy-ai
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:3000`

## 🚀 Deployment

The application is configured for automatic deployment to GitHub Pages:

1. Push changes to the `main` branch
2. GitHub Actions will automatically build and deploy
3. Access your live site at: `https://moeabdelaziz007.github.io/amrikyy-ai/`

## 🎨 Customization

### Colors
Update the CSS variables in `src/styles/App.css`:
```css
:root {
  --neon-green: #39FF14;
  --neon-blue: #00d4ff;
  --neon-purple: #b600ff;
  /* ... */
}
```

### Components
Each component is modular and can be customized independently. The quantum effects and animations can be adjusted in the respective component files.

### Services
Mock data and AI responses can be customized in `src/services/aiServices.js` to connect to real APIs or modify the quantum simulation parameters.
