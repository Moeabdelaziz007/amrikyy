Frequently Asked Questions

How do I update AuraOS to the latest version?

To update AuraOS, go to Settings > System > Updates. Click "Check for Updates" and if an update is available, click "Install Now". The system will automatically download and install the update. Make sure to save your work before updating.

How can I improve system performance?

To improve performance: 1) Close unused applications, 2) Clear cache regularly, 3) Disable startup programs you don't need, 4) Keep your system updated, 5) Use the System Monitor to identify resource-heavy processes.

How do I backup my data?

AuraOS provides automatic cloud backup through AuraCloud. Go to Settings > Backup & Sync to configure automatic backups. You can also manually backup files using the File Manager's export feature or use external storage devices.

What security features does AuraOS provide?

AuraOS includes: Built-in firewall, Real-time malware protection, Encrypted file system, Secure boot, Two-factor authentication, Privacy controls, and Regular security updates. All these features work together to keep your system and data safe.

How can I customize my desktop?

Customize your desktop through Settings > Appearance. You can change themes, wallpapers, icon sets, and color schemes. Right-click on the desktop for quick access to customization options. The App Store also offers additional themes and customization packs.
