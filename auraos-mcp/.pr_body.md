feat: add AuraOS MCP adapter with FastAPI

This PR adds a FastAPI-based MCP adapter to integrate AuraOS with Cursor IDE.

- GET /mcp/health
- POST /mcp/describe
- POST /mcp/list_workflows
- POST /mcp/restart_service
- POST /mcp/get_metrics
