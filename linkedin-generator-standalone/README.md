# 🚀 LinkedIn Viral Post Generator

أداة مجانية لتوليد منشورات LinkedIn فيروسية باستخدام الذكاء الاصطناعي

## ✨ المميزات

- 🔍 جلب آخر الأخبار والتريندات
- 🤖 توليد منشورات فيروسية بالذكاء الاصطناعي
- 🎨 اختيار نبرة المنشور (تحفيزية، تقنية، قصصية)
- 📋 نسخ المحتوى بنقرة واحدة
- 📱 تصميم متجاوب لجميع الأجهزة
- 🌐 يعمل على GitHub Pages مجاناً

## 🚀 نشر سريع على GitHub Pages

### الطريقة الأولى: نسخ الملفات
1. إنشئ مستودع جديد على GitHub
2. ارفع ملفات `linkedin-generator-standalone/`
3. فعل GitHub Pages من إعدادات المستودع
4. اختر المجلد الرئيسي كمصدر

### الطريقة الثانية: Fork المشروع
1. اعمل Fork لهذا المشروع
2. انتقل إلى Settings → Pages
3. اختر المصدر: Deploy from a branch
4. اختر branch: main
5. اختر folder: /linkedin-generator-standalone

## 🔧 إعداد APIs (اختياري)

للحصول على أخبار حقيقية ومنشورات محسنة:

### 1. GNews API
- سجل في [gnews.io](https://gnews.io)
- احصل على API key مجاني
- أضف المفتاح في إعدادات المستودع: `GNEWS_API_KEY`

### 2. OpenAI API
- سجل في [OpenAI](https://platform.openai.com)
- احصل على API key
- أضف المفتاح في إعدادات المستودع: `OPENAI_API_KEY`

## 🎯 كيفية الاستخدام

1. أدخل كلمة مفتاحية (مثل: AI, Blockchain)
2. اضغط "جلب الأخبار المتداولة"
3. اختر نبرة المنشور المناسبة
4. اضغط "توليد منشور فيروسي"
5. انسخ المحتوى وانشره على LinkedIn

## 💡 نصائح للنجاح

- استخدم كلمات مفتاحية محددة ومتداولة
- اختر النبرة التي تناسب جمهورك
- انشر في أوقات الذروة (9-11 صباحاً، 1-3 عصراً)
- تفاعل مع التعليقات لزيادة الانتشار
